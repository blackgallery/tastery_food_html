  1. css/animate.css - ( Animate Effec )
  2. css/hover.css - ( Hover Effect )
  3. css/owl.carousel.css - ( Carousel )
  4. css/slick.css - ( Carousel )
  5. css/slick-theme.css - ( Carousel Style)
  6. css/magnific-popup.css - ( Image Preview )
  7. css/bootstrap.min.css - ( BootStrap )
  8. css/normalize.css - ( Default Style)
  9. css/style.css - ( Template Style)
  10. css/responsive.css - ( Template Responsive Style )

Mailchimp Setup

Mailchimp setup is easy but mandatory step. If you already have Maichimp account & a list, just go to sign up forms for the particular list. Get form action URL from embedded forms section. Paste this in following section of script:

                                    $('.mc-form').ajaxChimp({
                                        url: '', //Enter your URL here
                                        callback: callbackFunction
                                    });

Steps

  1. Signup / Login on Mailchimp.
  2. Create a list.
  3. For this list, go to 'Signup Forms' section.
  4. Go to 'Embedded Forms'.
  5. Copy action url from the form.
  6. Open jquery.ajaxchimp.min.js in js Folder. Paste the action url in jquery.ajaxchimp.min.js file.
